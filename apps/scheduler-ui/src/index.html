Phases and Tasklist

08-Jun-2024

 

Client: Ultimate Renovations

 

 

Simple Project Management WebApp to mirror and improve upon current Project Management Excel Sheet used by Ultimate Renovations. See Image at the end of this document for more information on requirements.

 

Client’s current functionality includes:

  1. Create, View, Edit, and Delete resource assignments
  2. Simplicity, no training necessary
  3. Project Managers can easily see their jobs by colour when viewing all jobs
  4. Ability to add additional information such a comments and notes

 

Client also wants to add:

  1. More views
    1. View by project site
    2. View by project manager
    3. View by resource? (my suggestion)
    4. Month view? (my suggestion)
    5. Day view? (my suggestion)
  2. Reporting (likely mirrors views?)
  3. Ability to be more specific about times for those days when a resource is at more than one project site during a work-day
  4. Mobile friendly
  5. Web-based, not locally hosted
  6. More efficient use of screen space
